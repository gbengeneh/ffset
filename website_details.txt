Create a modern premium website for **FFSET Lounge**, an entertainment lounge currently located in **Akure**, with plans to expand to **Lagos** later.

FFSET Lounge sells premium and expensive wines, offers snooker games, PlayStation/console gaming, social hangouts, events, and gaming competitions.

Build the website as a clean, luxury, dark-themed, mobile-responsive platform with a premium lounge feel.

Use the following stack:

* Next.js
* TypeScript
* Tailwind CSS
* Framer Motion for animations
* Use reusable components
* Use dummy/static data for now
* No backend integration yet
* No payment integration yet
* Make the UI ready for future backend/API integration

Website pages/sections to create:

1. **Homepage**

   * Hero section with premium lounge branding
   * Short intro about FFSET Lounge
   * Call-to-action buttons:

     * Join Competition
     * View Wines
     * Book a Table

     * Premium Wines
     * Snooker

     * Events & Hangouts

   * Featured competition preview

   * Gallery preview

Design requirements:

* Use luxury dark theme
* Use colors like black, gold, wine red, and white
* Use smooth animations
* Use responsive layout
* Use clean typography
* Make it look premium, not like a basic template
* Use placeholder images where real images are not available
* Use reusable components such as:

  * Navbar
  * Footer
  * Hero
  * SectionTitle
  * ServiceCard
  * WineCard
  * EventCard
  * CompetitionCard
  * GalleryGrid
  * AdminSidebar
  * DashboardCard

Important:

* Do not include VIP Lounge because FFSET Lounge does not currently have that yet.
* Keep future expansion support in mind for Lagos branch.
* Add comments where backend/API integration will later be connected.
* Make the project easy to extend later with Laravel backend or API.
* Ensure the website runs without errors.
